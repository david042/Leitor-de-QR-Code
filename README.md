# Leitor de Código QR

leitor de Código QR e outros tipos de código de barras extremamente simples para Android

só escaneia o código e mostra o resultado em texto


**tipos suportados**

* QR_CODE
* DATA_MATRIX	
* UPC_A
* UPC_E
* EAN_8
* EAN_13
* CODE_39
* CODE_93
* CODE_128
* CODABAR
* ITF
* RSS14
* PDF_417
* RSS_EXPANDED
* AZTEC
